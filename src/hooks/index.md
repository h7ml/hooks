# 这是什么?

`@dext7r/hooks` 是一个基于 `React` 的自定义 `Hooks` 库，旨在提供一套方便、易用的针对不同场景的 `Hooks` 封装。该库涵盖了多种常见的前端请求操作，包括 `Axios`、`Puppeteer`、`Fetch`、`Request` 等，以帮助开发者更高效地处理网络请求和浏览器操作
